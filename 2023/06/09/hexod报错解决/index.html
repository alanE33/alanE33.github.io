Hexo d报错解决
6月 9, 2023

使用 hexo d 命令把博客内容推到github上的时候报如下错误：

解决方案

  1. 进入博客目录删除 .deploy_git 文件夹
  2. 执行命令 git config --global core.autocrlf false
  3. 重新 hexo clean && hexo g && hexo d
  4. 遇到仍然失败，怀疑网速问题，科学上网后问题解决。
